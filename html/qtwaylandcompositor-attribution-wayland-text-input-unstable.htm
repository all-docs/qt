Wayland Text Input Protocol, version unstable v2

Adds support for text input and input methods to applications.

Used in the Qt Wayland Compositor, and the Qt Wayland platform plugin.

The sources can be found in tqtc-qt5/qtwayland/src/3rdparty/protocol/text-input-unstable-v2.xml.
