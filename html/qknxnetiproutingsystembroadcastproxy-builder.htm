Detailed Description

The common way to create a routing system broadcast frame is:

QKnxLinkLayerFrame cemi = // build cEMI frame
auto netIpFrame = QKnxNetIpRoutingSystemBroadcastProxy::builder()
    .setCemi(cemi)
    .create();
