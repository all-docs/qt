Lipi Toolkit, version 4.0.0

Lipi Toolkit (LipiTk) is an open source toolkit for online Handwriting Recognition.

Optionally used in Qt Virtual Keyboard.

The sources can be found in tqtc-qt5/qtvirtualkeyboard/src/plugins/lipi-toolkit/3rdparty/lipi-toolkit.
