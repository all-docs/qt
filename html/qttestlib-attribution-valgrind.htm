Valgrind, version 3.14.0

An instrumentation framework for building dynamic analysis tools.

Used on Linux ond MacOS in the Qt Test module.

The sources can be found in tqtc-qt5/qtbase/src/testlib/3rdparty/valgrind_p.h and tqtc-qt5/qtbase/src/testlib/3rdparty/callgrind_p.h.
