Obsolete Members for CameraFocus

The following members of QML type CameraFocus are obsolete. They are provided to keep old source code working. We strongly advise against using them in new code.

bool isFocusModeSupported(mode)

This method is obsolete. It is provided to keep old source code working. We strongly advise against using it in new code.

Returns true if the supplied mode is a supported focus mode, and false otherwise.


bool isFocusPointModeSupported(mode)

This method is obsolete. It is provided to keep old source code working. We strongly advise against using it in new code.
