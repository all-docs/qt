Qt Quick PDF QML Types

This QML module contains types for handling PDF documents.

To use the types in this module, import the module with the following line:

import QtQuick.Pdf 5.15

PdfDocument

A representation of a PDF document

PdfLinkModel

A representation of links within a PDF document

PdfNavigationStack

History of the destinations visited within a PDF Document

PdfSearchModel

A representation of text search results within a PDF Document

PdfSelection

A representation of a text selection within a PDF Document
