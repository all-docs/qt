QOpenGLFunctions_1_2 Class

The QOpenGLFunctions_1_2 class provides all functions for OpenGL 1.2 specification. More...

Header: #include <QOpenGLFunctions_1_2>
qmake: QT += gui
Since: Qt 5.1
Inherits: QAbstractOpenGLFunctions

This class was introduced in Qt 5.1.

Detailed Description

This class is a wrapper for functions from OpenGL 1.2 specification. See reference pages on opengl.org for function documentation.

See also QAbstractOpenGLFunctions.
