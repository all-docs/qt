QKnxNetIpCrd Class

The QKnxNetIpCrd class stores the data block returned with a KNXnet/IP connection request frame. More...

Header: #include <QKnxNetIpCrd>
qmake: QT += knx

Detailed Description

A KNXnet/IP connection response data (CRD) structure encapsulates two substructures that contain both host protocol independent data and host protocol dependent data.

See also QKnxNetIpConnectResponseProxy, QKnxNetIpCrdProxy, and Qt KNXnet/IP Connection Classes.
