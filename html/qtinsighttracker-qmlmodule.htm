Qt Insight Tracker QML Types

To import the QML types into your application, use the following import statement in your .qml file:

import QtInsightTracker

InsightCategory

Attaching type for category property

InsightConfiguration

Provides an interface to the Qt Insight Tracker configuration

InsightTracker

Provides an interface to the Qt Insight Tracker
