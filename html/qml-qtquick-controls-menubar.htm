MenuBar QML Type

Detailed Description

MenuBar can be added to an ApplicationWindow, providing menu options to access additional functionality of the application.

ApplicationWindow {
    ...
    menuBar: MenuBar {
        Menu {
            title: "File"
            MenuItem { text: "Open..." }
            MenuItem { text: "Close" }
        }

        Menu {
            title: "Edit"
            MenuItem { text: "Cut" }
            MenuItem { text: "Copy" }
            MenuItem { text: "Paste" }
        }
    }
}

See also ApplicationWindow::menuBar.
