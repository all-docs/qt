Detailed Description

The application description contains information about an OPC UA application.

Member Type Documentation

enum QOpcUaApplicationDescription::ApplicationType

This enum type holds the application type.

ConstantValueDescription
QOpcUaApplicationDescription::Server0This application is a server.
QOpcUaApplicationDescription::Client1This application is a client.
QOpcUaApplicationDescription::ClientAndServer2This application is a client and a server.
QOpcUaApplicationDescription::DiscoveryServer3This application is a discovery server.
