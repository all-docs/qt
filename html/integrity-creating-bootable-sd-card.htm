Creating Bootable SD Card

With a bootable micro SD card your i.MX6 Quad board can run the INTEGRITY monolith application on the board. To create the bootable SD card:

  1. Format the SD card with a FAT32 filesystem.
  2. Save the demo.image file (built with mkimage in Building U-Boot Image) and the device tree blob file imx6q-sabrelite.dtb for the i.MX6 Quad board under your SD card root.

    Note: You will find the device tree blob file in your INTEGRITY installation, under the path /usr/ghs/int1144/platform-cortex-a9/imx6-sabrelite/imx6q-sabrelite.dtb.

  3. Set the SD card into your i.MX6 Quad board.
