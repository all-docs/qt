Wayland IVI Extension Protocol, version 1.9.1

"Wayland IVI Extension" is forked from IVI Layer Management to define a common set of APIs by wayland style protocol and provide reference implementation which can be loaded on Weston.

Used in the Qt Wayland Compositor, and the Qt Wayland platform plugin.

The sources can be found in tqtc-qt5/qtwayland/src/3rdparty/protocol/ivi-controller.xml and tqtc-qt5/qtwayland/src/3rdparty/protocol/ivi-application.xml.
