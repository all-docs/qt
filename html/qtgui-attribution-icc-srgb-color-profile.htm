sRGB color profile icc file

An ICC color profile for PDF/A-1b compatible PDF files.

Used in Qt Gui (Embedded into PDF/A-1b files generated by QPrinter/QPdfWriter).

The sources can be found in tqtc-qt5/qtbase/src/3rdparty/icc/sRGB2014.icc.
