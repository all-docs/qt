QKnxNetIpCri Class

The QKnxNetIpCri class stores the additional information needed by communication channels to fulfill a connection request. More...

Header: #include <QKnxNetIpCri>
qmake: QT += knx

Detailed Description

A KNXnet/IP connection request information (CRI) structure encapsulates two substructures that contain both host protocol independent data and host protocol dependent data.

See also QKnxNetIpConnectResponseProxy, QKnxNetIpCriProxy, and Qt KNXnet/IP Connection Classes.
