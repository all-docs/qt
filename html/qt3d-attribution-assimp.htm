Open Asset Import Library, version 5.0.1

assimp (Open Asset Import Library) is a portable open source library to import various well-known 3D model formats in a uniform manner.

Used for importing assets for use with Qt 3D

The sources can be found in tqtc-qt5/qt3d/src/3rdparty/assimp.
