enum QHelpSearchQuery::FieldName

This enum is obsolete. It is provided to keep old source code working. We strongly advise against using it in new code.

This enum type specifies the field names that are handled by the search engine.

ConstantValueDescription
QHelpSearchQuery::DEFAULT0the default field provided by the search widget, several terms should be split and stored in the word list except search terms enclosed in quotes.
QHelpSearchQuery::FUZZY1Terms should be split in separate words and passed to the search engine.
QHelpSearchQuery::WITHOUT2Terms should be split in separate words and passed to the search engine.
QHelpSearchQuery::PHRASE3Terms should not be split in separate words.
QHelpSearchQuery::ALL4Terms should be split in separate words and passed to the search engine
QHelpSearchQuery::ATLEAST5Terms should be split in separate words and passed to the search engine
