QKnxAddress Broadcast

Detailed Description

A group address is a 2-byte value that does not need to be unique. A device may have more than one group address.

See also QKnxAddress::Individual and QKnxAddress::Notation.

Member Variable Documentation

QKnxAddress Group::Broadcast

Represents a single KNX group address with the value 0x0000 that is reserved for the broadcast address that shall be exclusively used in the broadcast communication mode.
