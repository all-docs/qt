Unicode Character Database (UCD), version 26

The Unicode Character Database (UCD) is a set of files that define the Unicode character properties and internal mappings.

Qt Core uses data obtained from UCD files for working with characters and strings.

The sources can be found in tqtc-qt5/qtbase/src/corelib/text/qunicodetables_p.h and tqtc-qt5/qtbase/src/corelib/text/qunicodetables.cpp.
